## AI业务天赋速测系统 - 从零到上线部署指南

本指南假设你完全不懂代码，只需要按顺序操作即可。


### 第一步：把项目文件夹复制到桌面

在你的文件管理器中，找到这个文件夹：

    ai-business-test

把整个文件夹复制到桌面。确认里面有这些文件：

    ai-business-test/
      index.html
      style.css
      script.js
      netlify.toml
      netlify/functions/calculate.js

共 5 个文件，缺一不可。


### 第二步：注册三个免费账号

依次打开浏览器，注册以下账号（全部免费，不需要绑银行卡）：

**2.1 注册 GitHub**
打开 https://github.com ，点右上角 Sign up，按提示注册。记住你的用户名和密码。

**2.2 注册 Netlify**
打开 https://app.netlify.com ，点 Sign up，选择「Log in with GitHub」，授权即可。不要选邮箱注册，直接用 GitHub 登录最方便。

**2.3 注册硅基流动（SiliconFlow）获取 AI 密钥**
打开 https://cloud.siliconflow.cn ，注册账号。注册后：

1. 点左侧菜单「API 密钥」
2. 点「创建新 API 密钥」
3. 复制生成的那串 sk-xxxxxxxx 开头的密钥
4. 打开记事本，粘贴进去保存好，后面要用

新用户有免费额度，足够测试使用。


### 第三步：下载安装 GitHub Desktop

打开 https://desktop.github.com ，下载安装包，安装完成后用你的 GitHub 账号登录。


### 第四步：把代码上传到 GitHub

1. 打开 GitHub Desktop
2. 点左上角 File → Add Local Repository
3. 它会提示找不到仓库，点「create a repository」
4. 在 Local Path 那里，点 Choose，选择桌面上的 ai-business-test 文件夹
5. Name 填 ai-business-test
6. 点 Create Repository
7. 你会看到所有文件出现在 Changes 列表里
8. 在左下角 Summary 输入框输入「初始提交」
9. 点 Commit to main
10. 点右上角 Publish repository
11. 取消勾选「Keep this code private」（确保是公开仓库，Netlify 免费版需要）
12. 点 Publish Repository

完成后，你的代码就在 GitHub 云端了。


### 第五步：在 Netlify 部署网站

1. 打开 https://app.netlify.com ，登录
2. 点 Add new site → Import an existing project
3. 选择 GitHub
4. 在列表中找到 ai-business-test，点击选中
5. 部署设置页面，所有选项保持默认，直接点 Deploy site
6. 等待 1-2 分钟，部署完成后你会看到一个网址，类似 random-name-12345.netlify.app

先别急着打开，还有一步关键设置。


### 第六步：配置 AI 密钥（最关键的一步）

1. 在 Netlify 网站后台，点你刚部署的站点名称进入
2. 点顶部菜单 Site configuration（站点配置）
3. 左侧菜单找到 Environment variables（环境变量）
4. 点 Add a variable
5. 填写：
   - Key（变量名）：SILICONFLOW_API_KEY
   - Values（变量值）：粘贴你在第二步保存的那串 sk-xxxxxxxx 密钥
6. 点 Create variable 保存
7. 回到 Deploys（部署）页面，点 Trigger deploy → Deploy site 重新部署一次

等部署完成，打开你的网址，AI 评测功能就生效了。


### 第七步：验证一切正常

1. 在手机或电脑浏览器打开你的 netlify.app 网址
2. 答完所有题目，填写手机号，点提交
3. 你应该看到：AI 生成的个性化评语（不是固定模板文字），以及分班推荐

**验证保密性：**
按 F12 打开开发者工具 → 选 Network 标签 → 重新提交一次。你会看到一个请求发往 /.netlify/functions/calculate，返回的只有分数和评语，看不到任何评分算法。

**验证数据收集：**
登录 https://app.netlify.com → 进入你的站点 → 点顶部 Forms 标签 → 你会看到 quiz-submissions 表单。点进去就能看到每条提交记录的完整信息：手机号、所有答案、分数、天赋标签、分班结果。也可以点 Export 导出 CSV 到 Excel 查看。


### 常见问题

**Q：网站打开白屏？**
检查 index.html 是否在 ai-business-test 文件夹的根目录，不能放在子文件夹里。

**Q：提交后报错 500？**
大概率是 AI 密钥没配对。去 Netlify → Site configuration → Environment variables 检查 SILICONFLOW_API_KEY 是否正确。即使密钥有问题，系统也会自动降级为模板评语，不会完全崩溃。

**Q：Forms 里看不到数据？**
第一次提交后 Forms 标签才会出现。确保至少成功提交过一次测评。

**Q：想换题目内容？**
修改 script.js 文件中 basicQuestionsPool 和 advancedQuestionsPool 数组里的文字即可，改完在 GitHub Desktop 里 commit 再 push，Netlify 会自动更新。

**Q：想换网站域名？**
在 Netlify 后台 → Domain management → 可以免费改 xxxx.netlify.app 前缀，也可以绑定自己的域名。
